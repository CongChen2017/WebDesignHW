Fitting Process


For this example, some "true" parameters were chosen. Then, a synthetic dataset was generated from the model. After that, a standard way to fit a line to the data (assuming independent Gaussian error bars) used. To marginalize over some "nuisance parameters" and find an estimate of the posterior probability function for others, MCMC was used. After 500 steps walking, the results were plotted using the corner.py module.
